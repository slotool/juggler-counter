# Juggler Counter

6号機ジャグラー向けのiPhone対応カウンター&設定判別ツールです。

## Features

- 6号機ジャグラー機種に対応
- 総回転、BIG、REG、子役、単独ボーナス、チェリー重複をカウント
- 機種ごとの判別要素を加味した設定期待度表示
- 設定差の大きい要素を機種別に表示
- iPhone縦向き・横向きレイアウト対応
- localStorageによる端末内保存
- PWA用manifestとService Worker同梱

## Usage

`index.html` をブラウザで開くと使えます。

GitHub Pagesで公開する場合は、このフォルダ内のファイルをリポジトリにアップロードし、PagesのSourceを対象ブランチに設定してください。

## Note

判別結果は解析値・実戦値をもとにした推測です。遊技結果や収支を保証するものではありません。
